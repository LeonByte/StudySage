# Privacy Policy

## Data Collection and Usage

The AI Study Assistant Discord bot:

1. Does not store user messages or conversation history
2. Does not collect personal information
3. Only processes message content temporarily to generate responses
4. Does not share any data with third parties

## Technical Data

The bot temporarily processes:
- Message content when directly addressed or when using slash commands
- User IDs for response routing only (not stored)

All data processing is done locally and is not retained after responding.
